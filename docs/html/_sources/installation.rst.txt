Установка OmegaT
****************

На начало сентября 2018.
Скачайте OmegaT 4.1.5 update 1 Windows with 64-bit JRE с сайта https://omegat.org/de/download
Страницу нужно прокрутить вниз, чтобы найти нужную версию

Затем запустите процесс установки. При установке можно принимать все опции по умолчанию
